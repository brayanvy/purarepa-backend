🌽 Productos

Catálogo de arepas artesanales. Lectura pública sin autenticación; escritura restringida a ADMIN.

✅ Acceso público: GET /api/products/ y GET /api/products/{id}/ no requieren token. Ideal para mostrar el catálogo a visitantes no registrados.
MétodoEndpointDescripciónAuth
GET /api/products/ Listar productos activos (paginado) Público
GET /api/products/{id}/ Detalle de un producto Público
POST /api/products/ Crear producto ADMIN
PATCH /api/products/{id}/ Actualizar producto parcialmente ADMIN
PUT /api/products/{id}/ Reemplazar producto completo ADMIN
DELETE /api/products/{id}/ Eliminar producto ADMIN
GET /api/products/ Listar productos Público

Devuelve la lista paginada de productos. Los usuarios no autenticados y los clientes solo ven productos con is_active=true. Los administradores ven todos.

Parámetros de query

ParámetroTipoDescripción
pageintegerNúmero de página (default: 1)
page_sizeintegerResultados por página (default: 20)

Ejemplo cURL

bash 
# Sin token — solo productos activos
curl https://purarepa-api.onrender.com/api/products/

# Con token ADMIN — todos los productos incluyendo inactivos
curl https://purarepa-api.onrender.com/api/products/ \
  -H "Authorization: Bearer <admin_token>"
POST /api/products/ Crear producto ADMIN

Crea un nuevo producto en el catálogo. Requiere multipart/form-data si se incluye imagen.

Request body (multipart/form-data)

CampoTipoRequeridoDescripción
namestring✅Nombre del producto (máx. 100 chars)
product_typeenum✅TRADICIONAL | QUESO | CHOCLO | RELLENA | MORADA
descriptionstring✅Descripción del producto
pricedecimal✅Precio en COP (ej: 4500.00)
stockinteger✅Unidades disponibles (≥ 0)
is_activeboolean❌Visible en catálogo (default: true)
imagefile❌Imagen del producto (JPG, PNG, WebP)

Ejemplo cURL

bash 
curl -X POST https://purarepa-api.onrender.com/api/products/ \
  -H "Authorization: Bearer <admin_token>" \
  -F "name=Arepa de Choclo" \
  -F "product_type=CHOCLO" \
  -F "description=Arepa dulce de maíz tierno" \
  -F "price=4500.00" \
  -F "stock=50" \
  -F "image=@/ruta/a/imagen.jpg"

PATCH /api/products/{id}/ Actualizar producto parcialmente ADMIN

Actualiza uno o más campos del producto. Solo envía los campos que deseas modificar.

Ejemplo — actualizar stock y precio

bash 
curl -X PATCH https://purarepa-api.onrender.com/api/products/5/ \
  -H "Authorization: Bearer <admin_token>" \
  -H "Content-Type: application/json" \
  -d '{"stock": 100, "price": "4800.00"}'
