📊 Reportes

Datos analíticos de ventas, clientes y productos. Exclusivo para administradores.

🔒 Todos los endpoints de reportes requieren rol ADMIN. Devuelven 403 para usuarios con rol CLIENT.
MétodoEndpointDescripciónAuth
GET /api/reports/dashboard/ Top 5 productos y ventas mensuales ADMIN
GET /api/reports/sales/ Total de ingresos y número de órdenes pagadas ADMIN
GET /api/reports/clients/ Lista de clientes con actividad de compra ADMIN
GET /api/reports/dashboard/ Dashboard principal ADMIN

Devuelve los 5 productos más vendidos (por unidades) y el desglose de ingresos por mes, calculados sobre órdenes con estado PAID.

Respuesta — 200 OK

  -H "Authorization: Bearer <admin_token>"
